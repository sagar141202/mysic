# 🎵 Mysic

A premium glassmorphism music streaming web app built with React, Vite, and Tailwind CSS v4.

## Tech Stack

- **React** — UI framework
- **Vite** — Build tool and dev server
- **Tailwind CSS v4** — Utility-first styling
- **Framer Motion** — Animations and transitions
- **YouTube Embed API** — Music playback

## Features

- 🎨 Glassmorphism dark UI design
- 🎵 YouTube-powered music playback
- 📋 Playlist and song management
- 🔍 Search and filter songs
- ❤️ Favorites / liked songs
- 📱 Fully responsive (mobile, tablet, desktop)

## Getting Started

```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Build for production
npm run build
```
